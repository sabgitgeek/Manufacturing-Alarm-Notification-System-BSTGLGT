# Project-AlarmNotification-BSTGLANGSAT
Developed a web-based Process Monitoring &amp; Alarm Notification System that integrates with ASPEN and SQL Server to monitor manufacturing process data in real time. Automated threshold-based email alerts, configurable alarm settings, and trend visualisation to improve response time and reduce manual monitoring.

# Overview

A proof-of-concept implementation of an industrial process
monitoring application.

# Features

• Real-time monitoring
• SQL integration
• Configurable thresholds
• Email notifications
• Dashboard visualization

# Tech Stack

- ASP.NET
- C#
- SQL Server
- Chart.js
- SMTP
